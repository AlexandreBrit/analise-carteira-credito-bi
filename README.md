# 📊 Análise de Carteira de Crédito

Projeto de Business Intelligence desenvolvido para análise de desempenho, risco e oportunidades em carteira de crédito.

---

## 🎯 Objetivo

Analisar a carteira com foco em:

- Volume solicitado vs. utilizado  
- Pipeline ponderado (potencial de desembolso)  
- Distribuição por produto e gerente  
- Identificação de clientes com alta utilização  

---

## 🛠️ Tecnologias

- SQL (extração e transformação de dados)  
- Power BI (visualização e dashboard)  
- Excel / CSV (base de dados)

## 💡 Insights de Negócio

- Alta concentração de crédito no produto **Capital de Giro**, indicando dependência da carteira nesse segmento  
- O gerente **Ana** apresenta maior volume desembolsado, sendo o principal driver de resultado  
- Clientes com utilização acima de 80% representam oportunidade de **aumento de limite (upsell)**  
- Pipeline ponderado indica **potencial de crescimento da carteira**, considerando operações em análise  
- Produtos com alta utilização podem indicar tanto oportunidade quanto risco (necessidade de acompanhamento)  

---

## 🚀 Diferenciais do Projeto

- Construção de métrica de **pipeline ponderado** para projeção de desembolso  
- Análise integrada de **performance comercial + risco de crédito**  
- Estrutura completa: **dados → análise → visualização**  
- Aplicação direta ao contexto de instituições financeiras
